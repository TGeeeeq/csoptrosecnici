# Povinná publicita PPK (od 2026)

Od roku 2026 jsou projekty Programu péče o krajinu (PPK B) spolufinancovány SFŽP ČR
(výzva NPŽP č. 28/2025), a proto platí povinná publicita. Zdroj: Výzva PPK B č. 2/2026,
kap. 10 + Příručka pro žadatele PPK B, kap. 6.

## Co musí příjemce zveřejnit (web + alespoň 1 příspěvek na sociálních sítích)

1. loga MŽP, SFŽP ČR a AOPK ČR,
2. povinné sdělení: „Tento projekt je spolufinancován Státním fondem životního
   prostředí ČR na základě rozhodnutí ministra životního prostředí.",
3. hlavní cíle projektu,
4. termín ukončení projektu.

Splnění se dokládá výstřižky (screenshoty) k Žádosti o platbu.

Při zakládání nového krajinného prvku (výsadba, tůně, mokřady…) navíc:
- po dobu realizace plakát A3,
- poté po dobu udržitelnosti stálá deska / infotabule A3 (dle Grafického manuálu NPŽP,
  s logem AOPK ČR).

## Pravidla použití loga AOPK ČR (Grafický manuál AOPK 2026, PDF zde)

- barva Pantone 342 / CMYK 100-30-80-25; na světlém podkladu zelená varianta,
  na rušivém/tmavém podkladu bílá varianta nebo podkladový obdélník,
- ochranná zóna 1/5 šířky piktogramu kolem celého loga,
- zakázáno: deformace, změna poměrů, barev, písma či rozmístění prvků.
